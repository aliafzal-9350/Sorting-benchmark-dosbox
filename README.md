# Sorting Benchmark for DosBox

This repository contains a collection of sorting algorithms benchmarked within the DosBox environment. 

## Purpose
The purpose of this project is to evaluate the performance of various sorting algorithms under the constraints of the DosBox setup.

## Structure
- `bubble_sort.py`: Implementation of Bubble Sort.
- `quick_sort.py`: Implementation of Quick Sort.
- `merge_sort.py`: Implementation of Merge Sort.

## Getting Started
Clone this repository and set up your DosBox environment to run the benchmarks.
